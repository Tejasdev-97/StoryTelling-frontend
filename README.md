# 🚀 MARS — A Journey Beyond

An immersive, scroll-driven storytelling web experience that simulates a mission to Mars.  
Built using modern web technologies, this project combines animation, interactivity, and cinematic design to create a visually engaging narrative.

---

## 🌌 Overview

**MARS — A Journey Beyond** is a frontend experience that takes users through a fictional Mars mission:

1. **The Call** – Introduction to the mission  
2. **Ignition** – Rocket launch sequence  
3. **The Void** – Deep space journey  
4. **Red Horizon** – Mars landing  
5. **A New World** – Exploration on Mars  

The project emphasizes **UI/UX, animation, and storytelling**, making it ideal for showcasing frontend skills.

---

## ✨ Features

- 🎬 Scroll-based storytelling (multi-act experience)
- 🚀 Rocket launch animation with real-time HUD
- 🌌 3D starfield using Three.js
- 🎯 GSAP-powered smooth animations
- 🛰️ Interactive elements (probe launch, flag planting, rover movement)
- 🪐 Dynamic planetary visuals (Earth → Mars transition)
- 🎨 Custom cursor & immersive UI
- 📱 Responsive design
- ⚡ Performance optimized rendering

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Advanced animations & layout)  
- JavaScript (Vanilla JS)  
- GSAP (GreenSock Animation Platform)  
- ScrollTrigger  
- Three.js
